# EDITORIAL OPTIMIZATION SYNONYMS AND ANTONYMS
# Variations and opposites for video editing terminology

## Basic Editing Actions
"trim" → Synonyms: cut, shorten, clip; Antonyms: extend, lengthen, add to
"split" → Synonyms: divide, separate, cut; Antonyms: join, merge, combine
"join" → Synonyms: merge, combine, connect; Antonyms: split, separate, divide
"crop" → Synonyms: trim, clip, cut out; Antonyms: expand, extend, add
"adjust" → Synonyms: modify, fine-tune, tweak; Antonyms: leave, maintain, keep as is

## Transitions
"fade" → Synonyms: dissolve, blend, gradual transition; Antonyms: cut, abrupt change
"dissolve" → Synonyms: blend, fade, mix; Antonyms: cut, hard transition
"slide" → Synonyms: swipe, glide, shift; Antonyms: static, fixed
"pan" → Synonyms: move, sweep, tilt; Antonyms: fixed, stationary

## Text Elements
"overlay text" → Synonyms: on-screen text, text addition; Antonyms: remove text, text-free
"caption" → Synonyms: subtitle, text description; Antonyms: no text, without description
"title" → Synonyms: heading, name, label; Antonyms: unnamed, unlabeled

## Pacing Elements
"slow motion" → Synonyms: slowed down, reduced speed; Antonyms: fast motion, sped up
"fast motion" → Synonyms: sped up, increased speed; Antonyms: slow motion, slowed down
"timelapse" → Synonyms: accelerated footage, compressed time; Antonyms: real-time, normal speed
"rhythm" → Synonyms: tempo, beat, flow; Antonyms: arrhythmic, irregular

## Structure Components
"hook" → Synonyms: opener, intro, attention grabber; Antonyms: conclusion, outro
"intro" → Synonyms: opening, beginning, lead-in; Antonyms: outro, conclusion, ending
"outro" → Synonyms: ending, conclusion, closing; Antonyms: intro, opening, beginning
"teaser" → Synonyms: preview, sneak peek; Antonyms: full reveal, complete content

## Audio Elements
"voiceover" → Synonyms: narration, commentary, spoken word; Antonyms: silent, no narration
"background music" → Synonyms: soundtrack, BGM, score; Antonyms: silence, no music
"fade in" → Synonyms: gradually introduce, slowly add; Antonyms: fade out, abrupt start
"fade out" → Synonyms: gradually remove, slowly diminish; Antonyms: fade in, abrupt end

## Visual Elements
"filter" → Synonyms: effect, overlay, treatment; Antonyms: original, unfiltered, raw
"effect" → Synonyms: treatment, modification, enhancement; Antonyms: original, untreated
"overlay" → Synonyms: superimpose, layer on top; Antonyms: remove, bare, plain

## Quality Adjustments
"sharpness" → Synonyms: clarity, definition, crispness; Antonyms: softness, blur, haziness
"brightness" → Synonyms: luminosity, lightness; Antonyms: darkness, dimness
"contrast" → Synonyms: difference, distinction; Antonyms: similarity, uniformity
"stabilization" → Synonyms: steady, smooth out; Antonyms: shaky, unsteady

## Composition Techniques
"rule of thirds" → Synonyms: thirds guideline, composition grid; Antonyms: centered composition
"balance" → Synonyms: equilibrium, harmony, proportion; Antonyms: imbalance, disproportion
"symmetry" → Synonyms: mirror image, evenness; Antonyms: asymmetry, imbalance
"depth" → Synonyms: perspective, dimension; Antonyms: flatness, two-dimensionality

## Enhancement Strategies
"highlight" → Synonyms: emphasize, spotlight, feature; Antonyms: downplay, minimize
"focus" → Synonyms: emphasize, concentrate on; Antonyms: blur, de-emphasize
"enhance" → Synonyms: improve, upgrade, elevate; Antonyms: diminish, worsen, degrade
"optimize" → Synonyms: improve, perfect, refine; Antonyms: worsen, degrade, diminish
